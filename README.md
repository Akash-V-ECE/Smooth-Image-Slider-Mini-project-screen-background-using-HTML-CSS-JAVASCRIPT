Image Slider  
This project is an image slider built with HTML, CSS, and JavaScript. The slider displays images with descriptions and allows users to navigate between them using left and right arrow buttons. The design is optimized for a medium screen size with a background for the image slider.

Features:  
Image Slider: A responsive image slider that showcases six different images.
Navigation Buttons: The slider includes "Previous" and "Next" buttons for easy navigation between images.
Smooth Transitions: The images transition smoothly when navigating.
Animated Content: Each image displays a name, description, and "See more" button with a fade-in animation.

Technologies Used:  
HTML5: Structure of the webpage.  
CSS3: Styling and animations for the image slider.  
JavaScript: Functionality for the navigation buttons.  

Code Structure:  
index.html: Contains the structure of the image slider.
style.css: Includes all the styling for the image slider, including animations and layout.
app.js: Handles the functionality for the navigation buttons.

Customization:  
Image Replacement: Replace the images in the ./media/ directory with your own images. Ensure the filenames match those referenced in the index.html file.
Text Content: Update the name, des, and button elements within each .item div in index.html to customize the text displayed with each image.
